# Quantara 🏆

Application mobile de prédictions sportives intelligentes basée sur l'IA.

## Sports couverts
- ⚽ Football
- 🏒 Hockey sur glace
- 🏀 Basketball

## Structure du projet

```
Quantara/
├── mobile/      → Application Flutter (iOS & Android)
├── backend/     → Supabase (PostgreSQL + Edge Functions)
└── docs/        → Documentation complète du projet
```

## Démarrage rapide

1. Lire `docs/architecture/overview.md` pour comprendre le système global
2. Lire `mobile/README.md` pour setup le projet Flutter
3. Lire `backend/README.md` pour setup Supabase

## Modèle économique
- **Freemium** : accès basique gratuit, prédictions premium payantes
- **Paiement** : CinetPay (Wave, Orange Money, MTN Money, Moov)
- **Cible initiale** : Côte d'Ivoire, puis Afrique de l'Ouest
